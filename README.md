## Installation

+ Download
+ Open it on your IDE
+ Windows : *py -m pip install -r requirements.txt*
    + Linux : *pip install -r requirements*
+ Windows : *pip install Pillow tk*
     + Linux : *pip install Pillow tk*
+ *py gui.py* OR *py main.py*
